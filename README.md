# Memotest
**Memotest made with Html, Css and JavaScript. Using BootStrap for styles and Cypress for testing.** 

### Installation: 
- download or clone repo
- run: npm install  
- Install live-server
- run: npm run start
- for tests : 
	npm run cypress:open  | npm run cypress:run  
 
 ### Demo:  https://nahuelc00.github.io/memotest/
