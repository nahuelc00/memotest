# Memotest
**Memotest realizado con Html, Css y JavaScript. Usando BootStrap para estilos y Cypress para testing.** 

### Installation: 
- download or clone repo
- run: npm install  
- Install live-server
- run: npm run start
- for tests : 
	npm run cypress:open  | npm run cypress:run  
 
 ### Demo:  https://nahuelc00.github.io/memotest/
